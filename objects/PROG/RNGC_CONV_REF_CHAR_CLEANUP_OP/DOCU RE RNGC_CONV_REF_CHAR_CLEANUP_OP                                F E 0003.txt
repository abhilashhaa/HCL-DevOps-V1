U1 &PURPOSE&                                                               
AS Ce programme supprime les valeurs (entrées AUSP) des caractéristiques de
   référence pour les objets de gestion qui sont des nœuds ou des articles 
   configurables.

U1 &FEATURES&                                                              
AS Paramètres :                                                            
AS Langue : la langue indiquée permet de déterminer les valeurs de         
   caractéristiques objet dépendantes de la langue.                        
AS Journaliser dans fichier : si cette option est cochée, le journal       
   principal du programme est sauvegardé dans un fichier avec le nom du    
   programme. Si elle n'est pas cochée, le transport est enregistré à      
   l'aide d'un outil de journalisation classique.                          
AS Mode test : si cette option est cochée, les entrées AUSP ne sont pas    
   supprimées, mais elles sont répertoriées dans un spool.                 
AS Objet de gestion : si la zone est renseignée, le programme exécute      
   l'opération uniquement pour l'objet de gestion indiqué (cette zone peut 
   être un MATNR ou n'importe quel ID objet de gestion étant classifié). Si
   la zone n'est pas renseignée, le programme traite tous les objets de    
   gestion.
