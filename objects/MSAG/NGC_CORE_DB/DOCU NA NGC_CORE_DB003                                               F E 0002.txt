U1 &CAUSE&                                                                 
AS En raison de l'échec du système, il existe au moins un objet de gestion 
   pour lequel les valeurs de caractéristiques de référence ne peuvent pas 
   être récupérées par le module fonction OBJECT_CHECK. Le journal complet 
   du programme de conversion (RNGC_CONV_REF_CHAR) donne de plus amples    
   informations. Les ID des objets de gestion sont indiqués par des        
   messages d'avertissement.                                               
U1 &SYSTEM_RESPONSE&                                                       
AS Les valeurs de caractéristiques de référence de ces objets de gestion ne
   peuvent pas être copiées dans la table AUSP. Elles sont alors ignorées. 
U1 &WHAT_TO_DO&                                                            
AS Si les valeurs de caractéristiques de référence de ces objets de gestion
   ne sont pas tenues d'être copiées dans AUSP, cette action n'est alors   
   pas requise. Dans le cas contraire, contactez le support SAP.
