U1 &CAUSE&                                                                 
AS Incohérence de la base de données : Il existe au moins un               
   klart-obtab-objek qui a plusieurs CUOBJ dans la table INOB. Le journal  
   complet du programme de conversion (RNGC_XPRA_04_CLF_HDR) donne de plus 
   amples informations. Les CUOBJ concernés sont indiqués par des messages 
   d'avertissement.                                                        
U1 &SYSTEM_RESPONSE&                                                       
AS Dans ces cas, les premiers CUOBJ sont utilisés dans la table CLF_HDR.   
U1 &WHAT_TO_DO&                                                            
AS L’incohérence dans la base de données peut être éliminée en fusionnant  
   tous les CUOBJ dans le premier. Par conséquent, tous les autres CUOBJ   
   pour le klart-obtab-objek doivent être remplacés (KSSK, AUSP, etc.) par 
   le premier (qui est placé dans CLF_HDR) et ces autres CUOBJ doivent     
   alors être effacés d’INOB.
