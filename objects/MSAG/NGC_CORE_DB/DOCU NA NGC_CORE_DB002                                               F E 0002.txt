U1 &CAUSE&                                                                 
AS Il existe au moins un objet de gestion externe bloqué. Le journal       
   complet du programme de conversion (RNGC_CONV_REF_CHAR) donne de plus   
   amples informations. Les ID des objets de gestion bloqués sont indiqués 
   par des messages d'avertissement.                                       
U1 &SYSTEM_RESPONSE&                                                       
AS Les valeurs de caractéristiques de référence de ces objets de gestion ne
   peuvent pas être copiées dans la table AUSP. Elles sont alors ignorées. 
U1 &WHAT_TO_DO&                                                            
AS Si les valeurs de caractéristiques de référence de ces objets de gestion
   ne sont pas tenues d'être copiées dans AUSP, cette action n'est alors   
   pas requise. Dans le cas contraire, débloquez les objets de gestion     
   (d'autres applications sont actuellement en train de les bloquer) et    
   exécutez de nouveau la conversion.
